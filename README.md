# SQL Query Optimizer Web Service

A web-based SQL Server Query Optimizer that accepts SQL queries, analyzes their performance using built-in tools, and provides suggestions for optimization. Designed as a RESTful service using **Python Flask**, it assists developers and DBAs in identifying inefficient queries and improving overall database performance.

## Features
- Accepts SQL queries via web interface.
- Analyzes Execution Plans to detect bottlenecks.
- Utilizes SQL Server DMVs for performance insights.
- Provides optimization suggestions for better query performance.
- Lightweight Flask-based backend with a simple HTML frontend.

## Technologies Used
- Python (Flask)
- HTML / CSS (Frontend)
- SQL Server (Performance Analysis)
- DMVs & Query Execution Plans
